# 🏨 CISEC - Sistema de Gerenciamento para Hóspedes de Hotel

Bot do Telegram para gerenciamento de hóspedes e serviços de hotel.

## 🚀 Como rodar o projeto (Para Desenvolvedores)

### 1. Pré-requisitos
- Python 3.8+
- Conta no Telegram
- Bot criado via [@BotFather](https://t.me/botfather)

### 2. Instalação

```bash
# Clone o repositório
git clone https://github.com/CISECAM/TCC-CISEC.git
cd TCC-CISEC

# Crie ambiente virtual (recomendado)
python -m venv venv

# Ative o ambiente
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# Instale dependências
pip install python-telegram-bot python-dotenv
